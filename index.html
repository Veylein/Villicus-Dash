The Ultimate VIP Moderation Dashboard

Control every aspect of Villicus: features, staff roles (1–10), warn thresholds, tickets, roles, muted/quarantine roles, themes and more — all with a premium UX.

Themes

Quick Features

  • Toggle automod, antinuke, welcome, tickets, logging
  • Staff levels 1–10 mapped to roles
  • Warns → punishment thresholds with buttons
  • Theme presets, logo upload, and accent color

How it works

  1. Sign in with Discord (server-side OAuth) — website exchanges token for a short-lived guild-scoped JWT.
  2. Website calls bot API (JWT or X-API-KEY) to read/write `guild_settings` stored by `ModManager`.
  3. Commands in the bot check staff level & command-required level before executing moderation actions.

Publish

Push this folder to a GitHub repo and enable Pages (serve from `main` branch `/docs` or publish from this folder). For the interactive site, host the dynamic backend and update the Sign-in link to point at its `/login`.
